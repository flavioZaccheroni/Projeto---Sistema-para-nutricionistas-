# Release 1.0.0 - Nutri Clinic Pro

## Escopo

Primeira versao comercial do MVP desktop para nutricionistas, baseada nas fases 1
a 25 do manual de engenharia do sistema.

## Modulos disponiveis

- Login, usuarios e permissoes.
- Dashboard operacional.
- Pacientes, agenda, anamnese e triagem nutricional.
- Antropometria, composicao corporal, gasto energetico e exames laboratoriais.
- Diagnostico nutricional, planejamento alimentar, banco de alimentos, receitas e
  suplementos.
- Relatorios, financeiro, backup e seguranca.
- Aplicativo do paciente, portal web, IA assistiva e integracoes externas.
- Implantacao com checklist de release.

## Checklist antes da distribuicao

- Executar todos os testes automatizados.
- Abrir o app pelo `run_app.py` e autenticar com um usuario administrador.
- Executar o menu `Implantacao` > `Executar checklist`.
- Gerar o executavel Windows seguindo `docs/build_executavel.md`.
- Testar o executavel em uma maquina limpa ou usuario Windows separado.
- Criar backup do banco local antes de qualquer demonstracao comercial.
