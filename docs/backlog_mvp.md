# Backlog do MVP

## Alta prioridade

- Criar estrutura base do projeto.
- Implementar login e perfis.
- Modelar banco inicial.
- Criar cadastro de pacientes.
- Criar anamnese inicial.
- Criar antropometria básica com cálculos automáticos.
- Criar relatório clínico simples.

## Média prioridade

- Dashboard com indicadores.
- Agenda e consultas.
- Triagem nutricional.
- Exames laboratoriais.
- Evolução gráfica.

## Baixa prioridade

- Aplicativo do paciente.
- Portal Web.
- IA assistiva.
- Integrações externas.

## Critério de pronto por fase

- Especificação atualizada.
- Tela ou serviço implementado.
- Validações aplicadas.
- Testes executados.
- Documentação da fase atualizada.
